Growthings User Guide
======================================

Welcome to the User Guide of Growthings, an affordable, easily programmable table-top smart greenhouse for everyone!

If you are here for the first time and looking to get started on building your smart greenhouse, please go to the :ref:`installation` page for detailed guide on how to set up your hardware and software.

If you are looking for information on how start programming the electronics, please go to the :ref:`quickstart` page.

If you are trying to find out how to program a certain device, please click on the names of the sensors below the corresponding image in the table below.
